# Titanic Data Analysis

This project analyzes the Titanic dataset using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Project Tasks
- Data Cleaning
- Missing Value Handling
- Exploratory Data Analysis (EDA)
- Data Visualization
- Survival Analysis

## Insights
- Female passengers had higher survival rates
- Higher-class passengers survived more
- Age and family size affected survival
